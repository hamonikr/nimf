nimf-settings

nimf-settings — Tool for setting up Nimf

Synopsis

nimf-settings [OPTIONS...]

Description

nimf-settings is a GUI tool for configuring nimf.

Options

-h , --help

Show help messages

--help-all

Show all options

--help-gapplication

Show GApplication options

--gapplication-service

Enter GApplication service mode. Currently this option is used to trigger the nimf indicator.

Exit Status

0 on success, non-zero on failure
